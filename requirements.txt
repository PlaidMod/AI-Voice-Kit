# Python dependencies for Scout (AIY Voice Kit V1 voice assistant).
#
# Install with:  pip3 install -r requirements.txt
#
# NOTE: The `aiy` library (aiy.board, aiy.voice.audio, aiy.voice.tts) is NOT
# listed here. On the old AIY system image it was preinstalled; on a modern
# 64-bit Raspberry Pi OS you install it from the GitHub source into this venv.

# --- Google Gemini API (gemini-2.5-flash + Google Search grounding) ---
# Leave UNPINNED so pip pulls a recent SDK. Scout uses the NEW google-genai SDK
# (from google import genai), function calling, and google_search grounding.
# The older "google-generativeai" package is a different API and will NOT work.
google-genai

# --- Wake word "Hello Claude" (low-CPU, always-on) ---
pvporcupine        # wake-word engine
pvrecorder         # microphone capture that feeds Porcupine

# --- Speech-to-text now runs in the cloud via Gemini (see assistant.transcribe).
#     openai-whisper was removed: it was far too slow/inaccurate on a Pi 3B and
#     hogged the limited RAM. Nothing to install for STT. ---

# --- Speaker verification (only the owner gets answers). Pulls in torch,
#     librosa, webrtcvad. ---
resemblyzer

# --- Neural text-to-speech (clear voice, runs well on ARM). ---
piper-tts

# --- Saving opportunities to a Google Sheet ---
google-api-python-client
google-auth
google-auth-oauthlib

# --- Math/audio helper used throughout. numpy 2.x works fine on Python 3.13;
#     the old <2 pin has no wheel for Python 3.13 and forces a slow source
#     build that fails on ARM. ---
numpy

# --- Fallback text-to-speech, used only if aiy.voice.tts is unavailable ---
pyttsx3
